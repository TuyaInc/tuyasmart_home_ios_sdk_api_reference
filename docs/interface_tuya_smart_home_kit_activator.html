Detailed Description

TuyaSmartHomeKitActivator is used to pair HomeKit devices.

This class provides capabilities to pair HomeKit devices.

Adds HomeKit devices to a specified home.

The pairing token does not contain home information. You must call the method to get home information when you use getTokenSuccess:failure:.
