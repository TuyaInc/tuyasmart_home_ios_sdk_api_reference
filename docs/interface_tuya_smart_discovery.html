Detailed Description

TuyaSmartDiscovery is used for pairing in Wi-Fi Easy Connect (EZ) mode.

This class provides the EZ pairing capabilities for Wi-Fi devices.

Note
This EZ pairing process is different from the EZ mode in 'TuyaSmartActivator'. The pairing token does not contain home information. You must call the method 'bindDeviceWithHomeId:devIds:success:' to get home information after the devices are found.
